# Aspirant UI Kit

Recreates the aspirant-facing surface of Career Copilot: marketing landing, app shell (dashboard), notifications feed, AI chat, onboarding.

Open `index.html` for an interactive click-through.

## Components
- `MarketingLanding.jsx` — top-of-funnel page (nav, hero, exam-logo strip, features, testimonials, CTA)
- `DashboardShell.jsx` — command center (stats bar, exam targets, notifications, study plan, forum, profile)
- `NotificationsFeed.jsx` — full-screen notifications with filter bar
- `ChatPane.jsx` — AI career chat (starter chips, streaming shimmer, message list)
- `OnboardingStep.jsx` — multi-step profile builder (identity, education, attempts…)
- `Primitives.jsx` — Button, Pill, Card, Eyebrow, StatTile, etc.

All components lift tokens from `../../colors_and_type.css`.
